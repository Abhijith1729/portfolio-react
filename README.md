# Abhijith Portfolio

This is my personal developer portfolio built using React and Vite.

## Live Website
https://abhijith1729.github.io/portfolio-react/

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- GitHub Pages

## Features
- Responsive design
- Projects showcase
- Resume download
- Contact section

## Run Locally

Clone the project

git clone https://github.com/Abhijith1729/portfolio-react.git

Install dependencies

npm install

Run development server

npm run dev

Build project

npm run build

## Deployment

Deployed using GitHub Pages.
